# Generate Content Drafts

Read the approved strategy and the referenced input contract.

Create only the requested number of channel-specific drafts. Preserve language, audience, campaign claims, and zero-budget boundaries. Make every draft traceable to the approved strategy.

Do not approve, schedule, publish, or contact anyone. Return contract-valid drafts in `pending_approval` form so Tanaghom can enforce the human decision gate.
